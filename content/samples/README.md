# Sample policy documents

Use these Markdown files to try **Policy Compass** locally:

1. Copy or upload any file from this folder through the app UI (paste or file upload).
2. Optionally set `HF_TOKEN` in `.env` so passages are embedded with the open-source
   `sentence-transformers/all-MiniLM-L6-v2` model via the Hugging Face Inference API.

Files are fictional and intended for demos only.

| [nda-and-data-handling.md](./nda-and-data-handling.md) | Confidentiality, personal data, retention |
